1. 이미지 확장자
jpg, jpeg
- 1600만개 색상 사용
- 해상도 조절 가능

gif
- 256개 색상 사용
- 아이콘, 투명한 배경 이미지

png
-웹 브라우저상(네트워크 상)에서 사용하기 위해서
- gif 장점 + jpg 장점

2. 이미지 삽입
img 태그

- 속성
경로 : src (파일명.확장자) (다른 폴더라면 폴더명/파일명.확장자) (사이트에서 이미지 우클릭, 이미지 주소 복사)
설명 : alt
너비 : width
높이 : height

3. 음악 삽입
audio 태그

controls="controls" 라고 쓰기도 함
