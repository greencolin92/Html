select 태그
- 목록으로 보여주는 항목 중에서 하나 또는 여러개를 선택할 수 있도록 함
- option 태그
- optgroup으로 묶을 수 있음

html 문서 구조화
1. 공간 분할 태그
- div : 블록 형식으로 공간 차지 (줄바꿈)
- span : 인라인 형식으로 분할, 쓴 글자 만큼만 공간 사용

2. 시맨틱 태그
header : 페이지 제목, 페이지 소개
  - hgroup : 제목과 관련된 부제목을 묶어주는 역할

nav : 하이퍼링크를 보아둔 네비게이션 표현을 위한 태그
  - body 태그 안에서 어디든 ok
  - 본문 위치에 영향 받지 않음

aside : 왼쪽이나 오른쪽에 부수적인 내용이 들어가는 부분
  - 본문 흐름에 벗어나지 않는

section : 그룹으로 묶어주는 역할담당

article : 웹 페이지 상에서 실제 내용 의미
  - 본문과는 독립적인 콘텐츠 영역

footer : (꼬리말) 웹사이트의 저작권 정보, 저작권과 관련된 표기(꼬리말)
